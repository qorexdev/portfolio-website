## Features

- **Single-page Layout** — smooth scroll navigation between sections
- **Dark Theme** — dark background with green accent colors
- **Typing Effect** — animated hero section with typewriter text
- **Scroll Animations** — elements fade in via Intersection Observer
- **Fully Responsive** — works on desktop, tablet, and mobile
- **Zero Dependencies** — pure HTML, CSS, and vanilla JavaScript

## Sections

- **Hero** — intro with typing animation and CTA
- **About** — personal background and experience
- **Skills** — technical skills overview
- **Projects** — featured project showcase
- **Contact** — contact form and links

## Run Locally

Open `index.html` directly in a browser, or use a local server:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

Then visit `http://localhost:8000`.
