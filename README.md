# Portfolio Website

Personal developer portfolio — clean dark theme, responsive, built with pure HTML/CSS/JS.

## Features

- Single-page layout with smooth scroll navigation
- Dark theme with green accent colors
- Scroll-triggered animations via Intersection Observer
- Typing effect in the hero section
- Fully responsive design
- No frameworks or dependencies

## Run Locally

Simply open `index.html` in your browser, or use any local server:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Structure

```
├── index.html        # Main page
├── css/style.css     # Styles
├── js/main.js        # Interactions & animations
└── assets/           # Images and other assets
```
